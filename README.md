# How to quicly install AWS CLI on EC2 Ubuntu 18.04 Instance

```
# Install
sudo apt-get update
sudo apt-get install python-pip
pip install awscli --upgrade --user
sudo apt-get update
sudo apt-get install awscli
pip install awscli --upgrade --user
aws --version

# Setup Credentials
aws configure
```
